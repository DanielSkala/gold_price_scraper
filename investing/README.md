# Investment Projections

Simulates long-term S&P 500 investment growth with monthly contributions and configurable annual salary raises.

## Usage

```bash
python investing/investing.py
```

Generates a matplotlib chart and CSV (`sp500_investment_growth_with_raise.csv`) showing projected portfolio value over time.

## Parameters

Edit the function call in `investing.py` to configure:
- Monthly investment amount
- Annual interest rate (historical S&P 500 average)
- Investment duration (years)
- Annual raise percentage
